## Image Optimization Tips

Before adding images to production:

1. **Compress images** - Use tools like TinyPNG or Squoosh
2. **Use WebP format** - Better compression than JPG/PNG
3. **Responsive images** - Consider different sizes for mobile
4. **Alt text** - Already included in HTML, but verify accuracy
